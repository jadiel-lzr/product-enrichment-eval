       - Export `filterTestProducts(products: readonly Product[]): { kept: Product[], removed: Array<{ sku: string; reason: string }> }` -- iterates once, separating kept from removed with reason strings

    2. Create `enrichment/src/cleaning/normalizers.ts`:
       - Export `normalizeColor(product: Product): Product` -- returns `{ ...product, color: product.color.toLowerCase().trim() }`. Does NOT touch `color_original`.
       - Export `sanitizeTitle(product: Product): Product` -- returns `{ ...product, title: product.title.trim(), name: product.name.trim() }`

       - Export `generateCleaningReport(input: { totalInput: number, totalOutput: number, removed: Array<{ sku: string; reason: string }>, colorsNormalized: number, titlesTrimmed: number }): CleaningReport`

       - Export `CleaningResult` type: `{ cleanedProducts: Product[], report: CleaningReport }`
